## 如何截图

1. **运行项目**
   ```bash
   npm run dev
   ```

2. **访问** http://localhost:3000

3. **截图步骤**
   - Windows: Win + Shift + S
   - Mac: Cmd + Shift + 4
   - 或使用浏览器开发者工具截图

4. **保存位置**
   - 保存到 `public/` 目录
   - 命名为上述建议的文件名

5. **优化截图**
   - 建议使用 TinyPNG 压缩
   - 推荐宽度：1200px
   - 格式：PNG 或 WebP
